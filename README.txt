# 🤖 Advanced AI Chatbot

An AI-powered chatbot built using Node.js, Express, and OpenRouter API.  
This project demonstrates real-time AI integration, secure environment configuration, and a modern responsive UI.

---

## 🚀 Live Features

- ✅ AI-generated dynamic responses
- ✅ Conversation memory support
- ✅ Secure API key handling using `.env`
- ✅ REST API backend using Express.js
- ✅ Modern responsive frontend UI
- ✅ Ready for cloud deployment

---

## 🛠 Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js
- Axios

- OpenRouter API
- Model: `meta-llama/llama-3-70b-instruct`

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/dheeptha22/Advanced-ai-chatbot.git
cd Advanced-ai-chatbot
2️⃣ Install Dependencies
npm install
3️⃣ Configure Environment Variables

Create a .env file in root folder:

OPENROUTER_API_KEY=your_api_key_here
4️⃣ Start the Server
npm start

Open in browser:

http://localhost:3000

🌍 Deployment

This project can be deployed using:

Render

Railway

VPS Hosting

Make sure to add the environment variable in deployment settings:

OPENROUTER_API_KEY=your_key
