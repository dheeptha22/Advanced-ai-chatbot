
==============================
AI WEBSITE CHATBOT PROJECT
==============================

STEPS TO RUN:

1. Install Node.js from:
   https://nodejs.org

2. Extract this folder.

3. Open terminal inside project folder.

4. Install dependencies:
   npm install

5. Open server.js and add your OpenRouter API key:
   const API_KEY = "YOUR_OPENROUTER_API_KEY";

6. Start server:
   npm start

7. Open browser:
   http://localhost:3000

Your AI chatbot will run! 🚀
