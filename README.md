# 🤖 Advanced AI Chatbot

An AI-powered conversational chatbot built using Node.js, Express, and OpenRouter API.  
This project demonstrates backend API integration, secure environment configuration, and dynamic AI-generated responses with session-based conversation memory.

---

## 🚀 Project Overview

The Advanced AI Chatbot is a full-stack web application that allows users to interact with an AI model in real time. It integrates a Large Language Model (LLM) via API and maintains chat history during the session.

This project showcases practical skills in:

- API Integration
- Backend Development with Express.js
- Environment Variable Security
- Frontend-Backend Communication
- Version Control using Git & GitHub

---

## ✨ Features

- ✅ AI-generated real-time responses
- ✅ Session-based conversation memory
- ✅ Secure API key handling using `.env`

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/dheeptha22/Advanced-ai-chatbot.git
cd Advanced-ai-chatbot
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Configure Environment Variables

Create a `.env` file in the root directory and add:

```
OPENROUTER_API_KEY=your_api_key_here
```

⚠️ Important:
- Do NOT upload `.env` to GitHub.
- Keep your API key private.

### 4️⃣ Start the Server

```bash
npm start
```

Open your browser and visit:

```
http://localhost:3000
```
